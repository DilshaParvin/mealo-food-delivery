# Mealo 🍔

Mealo is a Django-based food delivery web application inspired by platforms like Swiggy and Zomato.

## Features
- Restaurant listing
- Food search functionality
- Food detail pages
- Add to cart system
- Order placement
- Restaurant dashboard
- Add/Edit/Delete food items
- Modern responsive UI

## Tech Stack
- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite

## Run Project

```bash
git clone https://github.com/DilshaParvin/mealo-food-delivery.git
cd mealo-food-delivery
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
